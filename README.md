# 🐱 핑크미야 (PinkMiya)

> 흑색과 분홍색으로 물든, 사랑스러운 반려동물을 위한 특별한 온라인 쇼핑몰

**핑크미야** 는 귀여운 고양이(그리고 반려동물) 친구들을 위한 화이트 & 핀크 컨셉의 풀스택 데모 쇼핑몰입니다.  
Flask + Jinja2 + Tailwind CSS로 만들어졌으며, 실제처럼 동작하는 로그인, 장바구니, 결제 플로우를 갖추고 있어요.

![PinkMiya Preview](https://via.placeholder.com/800x400/fce7f3/831843?text=PinkMiya+-+Cute+Pet+Shop)

## ✨ 주요 특징

- 🎀 **아기자기한 화이트+핀크 고양이 테마** — SVG 고양이 로고, 부드러운 파스텔 디자인
- 🛍️ **12종의 귀여운 상품** (침구, 장난감, 악세서리, 급식/위생, 의류, 간식)
- 🔗 **다양한 페이지 간 링킹** — 상세 페이지, 카테고리 필터, 관련 상품, 브레드크럼 등 완벽 연결
- 👤 **완전한 인증 시스템** — 회원가입, 로그인, 로그아웃 (Flask session)
- 🛒 **실제 동작하는 장바구니** — 수량 변경, 삭제, 합계 계산
- 💳 **결제 플로우** — 로그인 필수 → 배송 정보 입력 → 주문 완료 (예쁨 성공 페이지)
- 💖 **위시리스트 기능** — 하트 버튼으로 상품 추가/제거, 별도 페이지
- 📦 **프로필 & 주문 내역** — 가짜 주문 히스토리 포함
- 📱 **반응형 디자인** — 모바일에서도 완벽하게 예쁨
- 🚀 **단순 실행** — `python app.py` 한 줄로 바로 실행

## 🚀 로컬에서 실행하기

### 1. 저장소 클론
```bash
git clone https://github.com/JimProKing/pinkmiya.git
cd pinkmiya
```

### 2. 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. 실행
```bash
python app.py
```

브라우저에서 http://127.0.0.1:5000 접속!

### 테스트 계정
- 아무 이메일 + 4자 이상 비밀번호로 회원가입 / 로그인 가능
- 예시: `momo@catmail.com` / `123456`

## 🐾 재미있는 포인트

- 모든 상품이 **실제 고양이들이 좋아할 만한** 부드러운 디자인 컨셉
- 상품 상세에서 "비슷한 상품" 클릭 시 자연스럽게 페이지 이동
- 결제 후 귀여운 주문 완료 화면 + 주문번호
- 집사 후기 카드 (실제 사용자처럼)
- 장바구니가 비어있을 때도 예쁨 빈 상태 화면
